项目介绍

easy-cache

轻量易用的前端缓存工具，防止内存泄漏

简介

特色（推广的一些亮点）

安装下载

  • 下载地址 https://github.com/ANT-CYJ/easy-cache/releases
  • npm i fast-cache-npm
  • CDN https://unpkg.com/easy-cache-npm/release/bundle.js

快速使用

var EC = window.EasyCache;
var cache = new EC();
cache.set("a", 100);
alert(cache.get("a"));
